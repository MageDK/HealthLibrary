\Library

The main object for interacting with the Krames StayWell library. This object will handle all the requests for content from the library.

The Library will use a file cache to store the responses from the library. The Library will serve up the content from the Cache if the content is recent and has not expired. By default the cache expiration is set to 30 minutes (1800 seconds).
